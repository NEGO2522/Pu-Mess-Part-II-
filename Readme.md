# PU Mess Connect
### Poornima University Digital Dining Interface

PU Mess Connect is a mobile-optimized digital pass system designed to streamline the dining experience. It provides a secure and efficient alternative to physical identity cards for campus meal services.

---

## Core Features

### Instant Digital Identity
The platform generates a persistent, high-fidelity QR code linked to the student profile. This serves as the primary digital credential for mess entry, ensuring fast and touchless verification.

### Automated Meal Context
The interface dynamically updates based on the time of day, identifying the current meal session. This feature assists in coordinating entry during Breakfast, Lunch, Snacks, and Dinner windows.

### Live Service Schedule
A dedicated timings dashboard provides real-time status of mess operations. Active meal windows are highlighted with visual indicators to keep users informed of service availability.

### Verified Entry History
Users can access a comprehensive log of their meal history. This provides a transparent, date-stamped record of all verified entries across different mess halls on campus.

---

## Interface & Navigation

* **High-Clarity Design:** Utilizes a modern frosted-glass aesthetic for better readability in different lighting conditions.
* **Mobile-First Layout:** Designed for one-handed operation and rapid navigation between the home, history, and schedule views.
* **Service Accuracy:** Synchronized with official university mess timings for precise schedule reporting.
